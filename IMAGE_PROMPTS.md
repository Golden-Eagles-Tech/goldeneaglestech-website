professional images for the website. Save generated images to `assets/images/` with the

## Usage Notes

1. **Gemini Image Generation:** Go to gemini.google.com → "Generate images" and paste the prompt.
   Choose the highest resolution available.

2. **Alternatively use:** Adobe Firefly, Midjourney, or DALL-E 3 with the same prompts.

3. **After generating:** Save files to `website/assets/images/` with the exact filename shown.
   The website references these paths automatically.

4. **Optimization:** Run images through squoosh.app or TinyPNG before uploading to keep
   file sizes under 300KB for performance.
